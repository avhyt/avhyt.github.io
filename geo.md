## 几何证明

【题 1】
如下图，平行四边形 ABDC，点 C 在 BE 上，点 E 是 AF 中点。求证：BE // DF
![geo1](img/geo1.jpg)

【解 1】
连接 AD 交 BC 于点 G
$$\because 平行四边形 ABDC$$<br>
$$\therefore AG = DG$$<br>
$$即: G 是 AD 中点$$<br>
$$又 \because E 是 AF 中点$$<br>
$$\therefore EG 是 \triangle AFD 中位线$$<br>
$$\therefore EG // DF$$<br>
$$即: BE // DF$$
